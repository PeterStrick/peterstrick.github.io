Nitro Sniper GUI Help

The Nitro Sniper GUI User Interface is rather simple and contains of 4 Main Buttons.

  • The Main Menu Button, allows you to Start and Manage Nitro Snipers, aswell as their Settings.

  • The Check for Updates Button, allows you to check for Updates on the fly.

  • The License and About Button, allows you to see License and more Information about the NitroSniperGUI and it´s Nitro Snipers.

  • The Help Button, opens this Website where you can look for Help on how to use the NitroSniperGUI or a Nitro Sniper.



The Main Menu consists of the following Sub Menu Items:

  • The Exit Button, allows you to Exit and Close the Nitro Sniper GUI, aswell as all Nitro Snipers.

  • The Nitro Sniper Buttons, allows you to start it´s corresponding Nitro Sniper.

  • The Window Controls Button, allows you to minimize and maximize all Nitro Snipers, that are currently opened.

  • The Settings Button, opens the Settings Window, where you can tweak all sorts of Settings. Press here to find out more about the Settings.
