Nitro Sniper GUI Help

Main Settings Help:

  • The Token Setting, allows the Nitro Sniper to log into your Discord Account, in order to snipe Discord Nitro. Don't worry, it won't get sent to anyone else.

  • The Max Nitro to Snipe Setting, changes how much Nitro the Sniper should Snipe, before it goes into the Cooldown state.

  • The Cooldown in Hours Setting, changes how long the Sniper stays inside the Cooldown state for the above Setting.

  • The Snipe on Main Token Setting, changes if the Nitro Sniper should Snipe on the Main Token or Alt Tokens. You can specify Alt Tokens manually in the Config File under, Edit manually > Edit Go-Nitro Sniper Config

  • The Enable Privnote Sniper Setting, changes if the Nitro Sniper should Snipe Privnotes. Privnotes are messages that once opened (or sniped in this case) will self-destruct themselves.

    Information: The Nitro Sniper could crash sometimes if you enable this option.
