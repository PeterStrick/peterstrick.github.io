Sound Settings Help:

  • The Enable Sound Setting, changes if the Nitro Sniper should play a Sound if it applied a Nitro Code.

    Unchecking it will replace sound.mp3 in the Directory of the Nitro Sniper with a silent, short Sound File.

  • The Change Sound Menu, allows you to either change the Sound to a custom MP3 File, or a set of 6 Presets.

  • The Play/Stop Button, allows you to play/stop the Sound File.
